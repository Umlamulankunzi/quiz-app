Quiz Wiz API Info

Welcome to the Quiz Wiz API! Our API allows developers to integrate Quiz Wiz quizzes into their own applications, offering a wide range of quiz topics and interactive features. Whether you're building an educational tool, a trivia app, or any other project that could benefit from quizzes, our API provides a simple and powerful way to access and manage quiz data.

Getting Started

  1. Register for an Account

    To access the API, you need to register an account on Quiz Wiz. Here's how:

    • Sign Up: Create an account on Quiz Wiz.
  2. Sign up with a username, email and Password

    Once you have signed up successfully, you will be able to access the API.

    Note: Keep your login credentials secure. Do not share them publicly or expose them on your frontend code.

  3. Explore the API Documentation

    Familiarize yourself with the available endpoints and how to use them by visiting our comprehensive API documentation:

    View API Documentation

API Endpoints

  • API root:
    • GET /api/v1/
  • List categories: Retrieve a list of question categories
    • GET api/v1/categories/
  • View Category: View a single category
    • GET categories//
  • View Questions: View Questions under a category
    • api/v1/categories//questions/
